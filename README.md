News Aggregator (Flask)

Quick steps:
1. python -m venv venv
2. source venv/bin/activate  (Windows: venv\\Scripts\\activate)
3. pip install -r requirements.txt
4. python db_init.py
5. python app.py
6. Visit http://127.0.0.1:5000
